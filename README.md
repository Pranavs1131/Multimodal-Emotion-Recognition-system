# Multimodal Emotion Recognition System

A Multimodal Emotion Recognition System that detects human emotions using three different modalities:

- Facial Expressions (Camera Input)
- Speech Audio
- Text Sentiment Analysis

The system combines **Computer Vision, Speech Processing, and Natural Language Processing** techniques to improve emotion prediction accuracy.

---

## Features

- Real-time facial emotion detection using webcam
- Speech-based emotion classification from audio signals
- Text emotion detection using NLP techniques
- Multimodal fusion of predictions from vision, audio, and text
- Machine Learning and Deep Learning based approach

---

## Technologies Used

- Python
- TensorFlow
- OpenCV
- NLP (Natural Language Processing)
- Speech Processing
- MFCC Feature Extraction
- Deep Learning (CNN)

---

## System Architecture

The system processes three different inputs:

1. **Facial Expression Analysis**
   - Captures video using webcam
   - Detects faces using OpenCV
   - Uses a CNN model to classify emotions

2. **Speech Emotion Recognition**
   - Extracts audio features using **MFCC**
   - Classifies emotions using deep learning models

3. **Text Emotion Detection**
   - Performs sentiment and emotion analysis using NLP techniques

The outputs from these models are combined to generate the final emotion prediction.

---

## Workflow

1. Capture input (Camera / Audio / Text)
2. Preprocess the data
3. Extract features (Image / Audio / Text)
4. Run trained ML/DL models
5. Combine predictions from multiple modalities
6. Output final predicted emotion
